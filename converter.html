How to Use

Select a single-letter PNG image of an ASL alphabet gesture. Preview it, then press Convert into Text. The detected letter will appear below.

When you connect a real model / API later, replace the demo `convertImage()` implementation in the script with a POST to your backend.

Image Requirements

Use a clear PNG with a plain background. Center the hand in the frame. Avoid motion blur. Higher contrast = better results.

Privacy

Preview happens locally. No data is uploaded unless your backend integration sends it — this demo does not send files anywhere.
