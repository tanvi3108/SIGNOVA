Project synopsis (Signova)

Signova converts sign language to text. This front-end records short 5-second clips for each gesture attempt and sends them to the model server for prediction. The UI shows the returned label(s) and confidences.

Backend contract

POST to the configured endpoint with FormData field named video. Example server response:

{ "label":"HELLO", "scores":[{"label":"HELLO","score":0.92}, {"label":"YES","score":0.03}] }
If your backend uses a different field name or returns another schema, tell me and I'll adapt the parsing code.
