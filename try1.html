Signova — Frontend Demo

Bridging hands and words — accessible sign language to text UI

Signova is a friendly interface for a sign-language recognition system. This frontend demonstrates a clean, responsive UI you can connect to any model/backend later. Focused on accessibility, clarity and real-time feedback — built using only HTML, CSS & JavaScript.

About Signova (Frontend)

This frontend demonstrates the user-facing design for an AI sign-language reader. It intentionally avoids any vendor libraries so you can learn how HTML, CSS and JS work together. The Live Translate modal simulates a continuous prediction stream so the UI behaviour is visible during demo day.
